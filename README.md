# Portfolio Website

Welcome to my professional portfolio website! This site showcases my projects, skills, and experience.

## Features

- Responsive design that works on all devices
- Dark theme with beautiful colors
- Project showcase with descriptions and technologies used
- Skills section with visual progress indicators
- Contact information and social links
- CV download functionality

## Setup Instructions

1. Clone this repository:
   ```bash
   git clone https://github.com/yourusername/your-portfolio.git
   ```

2. Replace placeholder images with your own:
   - `profile.jpg` - Your profile photo
   - `logo.png` - Your personal logo
   - `project*.jpg` - Images for your projects

3. Update the CV.pdf file with your resume

4. Update the content in index.html with your personal information

5. Deploy to GitHub Pages:
   - Go to your repository settings
   - Scroll down to the "GitHub Pages" section
   - Select the source (usually the main branch)
   - Save the changes

## Customization

- Edit `index.html` to update content, projects, and contact information
- Modify `styles.css` to adjust colors and styling to match your preferences
- Update `script.js` to add additional functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive design principles
- Modern CSS features (Flexbox, Grid, CSS Variables)
